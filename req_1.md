生成一个任务信息查询页面, 主要包括：
1. 过滤筛选功能：根据任务状态、任务类型、提交用户等条件进行过滤。
2. 任务列表展示, 需要展示的字段及说明如下：
   - 任务ID, id
   - 任务类型, task_type
   - 提交用户, user
   - 任务状态, state
   - 创建时间, create_at
   - 开始执行时间, start_at
   - 完成时间, finish_at
   - 任务结果, result 字段为 JSON 格式文本，不直接在列表中显示内容， 提供 "查看结果" 功能。
   - 错误信息, error_message 字段为文本，不直接在列表中显示内容， 提供“错误详情”功能。
3. 分页功能：支持分页显示任务列表，默认每页显示 20 条记录, 支持跳转到指定页码，支持选择每页显示记录数（10, 20, 50, 100）。
4. 任务结果查看功能：点击“查看结果”按钮，弹出对话框显示任务结果的 JSON 格式内容，支持格式化和折叠查看。
请使用 Vue 3 和 Element Plus 组件库，且仅使用 Options API 来实现该页面。
